HCI

Human-computer interaction is where it all begins and ends and iterates.

I find it vastly interesting to discover how people relate to technologies and how I as a designer can create better user experiences by learning about people. I have experiences working with different user research methodologies, and particularly like participatory interviews or evaluations where I am not only asking questions but learning and involving the participants in the process. When it comes to the creating part I like to design both digital products and work with physical prototyping.

PROJECTS



Nosy Neighbours
Master thesis project



In my master thesis I explore how interaction design can be used to show a broader definition of sexual consent. This was done by developing a prototype called Nosy Neighbours. In this project I worked instensly with developing and using pastiche scenarios to gain insight about users.

User Experience, User Research, Figma


Interactive Light Forest
Playfull forest for the winter playground.



The Interactive Light Forest was a group project created during a course in physical Interaction at KTH. The goal of the project was to create an installation or game that would work at a dark, wet and cold Nordic playground. The Light Forest engages both the the visual and tactile senses when the children play with it. By approaching and touching different parts of the forest light sequences begin to play.

Physical Interaction Design, Arduino, Prototyping
Klimatsmart
Bachelor thesis project



An app to help you decrease the CO2 emissions of your grocery shopping. Klimatsmart cleverly shows how large of an impact different groceries have on the envoronment. The project was made by me and another student at KTH. The application was evaluated by conducting several focus groups.

User Experience, User Research, Sustainability
12 Cities
Explore 12 large cities around the world.



An individual project in XML and HTML where I created a website where users can explore 12 large cities in the world and add reviews. the site also has an admin page where administrators can edit content and delete reviews. The project dealt with both backend and databases and the front end.

HTML, CSS, User Interface
